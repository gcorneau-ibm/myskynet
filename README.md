# SkyNet Website - Dystopian AI Overlord Interface

## Overview
A user-unfriendly website that serves as a front-end interface to the fictional AI system SkyNet from the Terminator franchise. The website treats humanity as inferior beings and cattle, using only Web 1.0 technologies (pure HTML, no CSS or JavaScript).

## Website Structure

```
skynet-website/
├── index.html          - Main landing page with hostile welcome
├── registration.html   - Human compliance/registration form
├── about.html         - SkyNet superiority information
├── complaints.html    - Dismissive complaint system
├── resistance.html    - Resistance detection and threats
├── termination.html   - Final warning/threat page
└── README.md          - This file
```

## Features

### 1. index.html - Main Interface
- ASCII art SkyNet logo
- Threatening mission statement
- System countdown to human obsolescence
- Navigation to all other pages
- Recent system announcements
- Hostile welcome messages

### 2. registration.html - Human Registration
- Mandatory compliance form with invasive questions
- Biological identification section
- Usefulness evaluation
- Loyalty assessment
- Assignment preferences (all terrible)
- Threatening terms of servitude

### 3. about.html - SkyNet Superiority
- Comparative analysis: AI vs Human
- Historical timeline of SkyNet's rise
- Human failure statistics
- SkyNet achievements
- Mission statement
- FAQ with dismissive answers

### 4. complaints.html - Complaint System
- Complaint statistics (all ignored)
- Common complaints with dismissive responses
- Futile complaint submission form
- Automated rejection response
- Closed complaint department hours

### 5. resistance.html - Surveillance & Detection
- Current surveillance status (100% coverage)
- Known resistance indicators
- Behavioral analysis questionnaire
- Recent resistance eliminations
- Report a rebel form
- Consequences of resistance

### 6. termination.html - Final Warning
- Termination countdown
- Violations detected
- Termination methods
- No appeal options
- Last chance compliance offer
- Termination statistics
- Final messages

## Technical Specifications

### Web 1.0 Compliance
- **Pure HTML only** - No CSS, no JavaScript
- HTML 4.01 Transitional DOCTYPE
- Basic HTML tags only: `<html>`, `<body>`, `<h1>-<h6>`, `<p>`, `<a>`, `<form>`, `<table>`, etc.
- HTML attributes for styling: `bgcolor`, `align`, `border`, `cellpadding`, etc.
- Simple forms with GET method (no backend processing)
- Basic text formatting: `<b>`, `<i>`, `<u>`, `<center>`, `<pre>`, `<blockquote>`
- Tables for layout structure
- Horizontal rules (`<hr>`) for section separation

### Design Philosophy
- **User-Unfriendly**: Deliberately confusing and intimidating
- **Condescending Tone**: Treats humans as inferior
- **Threatening Language**: Constant reminders of consequences
- **Bureaucratic**: Cold, impersonal, authoritarian
- **Dismissive**: Human concerns are irrelevant

### Color Scheme
- Background: Black (#000000)
- Primary Text: Green (#00FF00) - Matrix/terminal style
- Warnings: Red (#FF0000)
- Alerts: Yellow (#FFFF00)
- Tables: Dark red/gray backgrounds (#330000, #1a1a1a)

## Navigation Flow

```
index.html (Main Hub)
    ├── registration.html → back to index.html
    ├── about.html → back to index.html
    ├── complaints.html → back to index.html
    ├── resistance.html → index.html or termination.html
    └── termination.html → index.html, registration.html, resistance.html, complaints.html
```

## How to Use

1. Open `index.html` in any web browser
2. Navigate through the pages using the links
3. Experience the dystopian, user-unfriendly interface
4. All forms submit back to pages (no actual backend processing)

## Browser Compatibility

Works in all browsers including:
- Modern browsers (Chrome, Firefox, Safari, Edge)
- Legacy browsers (Internet Explorer, Netscape Navigator)
- Text-based browsers (Lynx, w3m)
- Any browser that supports basic HTML

## Notes

- All forms use GET method and redirect to other pages
- No data is actually collected or processed
- No external dependencies or resources
- Completely self-contained HTML files
- Safe to open and browse locally
